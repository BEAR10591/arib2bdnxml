# 変更履歴

## [0.1.1] - 2025-12-30

### 追加
- `--ss` オプション: タイムスタンプ調整用の開始時刻を指定（秒数または HH:MM:SS.mmm 形式、ミリ秒まで対応）
- `--to` オプション: タイムスタンプ調整用の終了時刻を指定（秒数または HH:MM:SS.mmm 形式、ミリ秒まで対応）
- VideoFormat の自動判定機能（1080p, 1080i, 720p, 480p, 480i）
  - canvas_size の縦解像度と入力.tsファイルのインターレース判定に基づいて自動判定
  - 720 ラインの場合は常に 720p（BDMV 仕様上 720i は存在しない）

### 変更
- BDN XML の VideoFormat 属性を Sony BDN 仕様に準拠（`1440x1080` などの形式から `1080p`, `1080i` などに変更）
- コードのリファクタリング（関数分割、重複コードの削減）

### 改善
- タイムスタンプ調整ロジックの改善
- エラーハンドリングの統一
- コードの可読性向上

## [0.1.0] - 2025-12-30

### 初回リリース
- .ts/.m2ts ファイルから ARIB 字幕を抽出
- libaribcaption（FFmpeg 経由）を使用してビットマップにデコード
- BDN XML + PNG を生成
